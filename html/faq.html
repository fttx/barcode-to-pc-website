Frequently asked questions

Common issues

How do i put the scans inside my spreadsheet (or other software)?

  1. Download Barcode to PC server
  2. Install and start the server in your computer
  3. Install the app in your smartphone and open it
  4. Now the app should connect automatically to the server, if nothing is happening follow the instrucitons on screen
  5. Once the app is connected open your spreadsheet or whatever software you need to scan in, and position the cursor where you want to scan
  6. Scan your barcodes by tapping the camera icon in the app
  7. Thanks to the keyboard emulation, the barcode will be typed in the cursor position

Can i use it whitout a Wi-Fi connection?

If you don't have access to a Wi-Fi network, you can always use your smartphone hotspot, by following this steps:

  1. Enable the Wi-Fi hostspot on your smartphone See how here
  2. Connect your computer to the Wi-Fi hotspot
  3. Add the server manually using the QR Code


Note that Barcode to PC server doesn't require an internet connection, as matter of fact that it uses the LAN to communicate, thus, in most of the cases it is required only the Wi-Fi network.

Connection issues

The app doesn't find the computer

Make you sure that:

  1. Your computer and your smartphone are connected to the same network.
    It doesn't matter if your computer has a wired connection as long as it is plugged into the same access point where your smartphone is connected.
  2. Both app and server are updated to the latest version.
  3. Barcode to PC server is in your firewall and/or antivirus exceptions
  4. Your computer is discoverable (only Windows users). See How to set Windows discoverable

How do i configure my firewall?

You must add the Barcode to PC server executable to the firewall exceptions, and allow the connections for the port 57891 TCP and 5353 UDP.
If you don't know how to do it, see the tutorial for your OS:

  • Windows
  • macOS
  • Linux

Add the server manually

There are two ways to add the server manually.
The first is by using the QR Code:

  1. Open the server
  2. Click Info, and the connect QR Code should appear
  3. Open the app
  4. Open the menu and select the "Select server" page
  5. Tap the scan icon [ ] and scan the QR Code

The second one:
  1. Open the server
  2. Open the menu and select the "Select server" page
  3. Tap the add icon + and type the IP address of the server

The app shows the "Connection problem" error

The most common cause of this issue is that the firewall and/or antivirus is blocking the communication between the app and the server. See How do i configure my firewall?

Which port does the server use?

Barcode to PC server runs a WebSocket server on the 57891 TCP port.
In addition to that, Barcode to PC server announces its service through the 5353 UDP port.

The latter is not essential, but it allows the app to discover the server without the user action. If you don't allow connections through the 5353 UDP, you can still use the server and the app, but you will need to add the server manually

The server shows the "Apple Bonjour is missing" error

This means that the installer has failed to install the Apple Bonjour package and this implies that the app may fail to detect the server automatically, and in order to connect the app you may need to add the server manually.
To remove this alert and take advance of the zeroconf feature, try to install Barcode to PC server again with an administrator account and then reboot your system.
